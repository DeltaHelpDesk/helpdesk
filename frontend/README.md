# Install project
`yarn install`

## For local backend
**/frontend/.env** set: 
`ENDPOINT=http://localhost:3000`

## For remote backend
**/frontend/.env** set: 
`ENDPOINT=https://delta-helpdesk.herokuapp.com`

# Start project 
`yarn start`

# Build
`yarn build`
